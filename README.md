## Getting Started

Each project is fully self-contained. To run any of them locally:

```bash
# Clone the repository
git clone https://github.com/Lechguer/HTML-CSS-JS-projects.git

# Navigate into any project folder and open index.html
cd "HTML-CSS-JS-projects/Quiz-Game"
open index.html
```

No build step, no package manager, no setup needed.
